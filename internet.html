Internet et Web

  1. Que désigne ‘Le Web’ ?
    Le web, Le Web désigne un système permettant d'accéder et de partager des informations via Internet. Il repose sur des pages interconnectées par des liens et est accessible grâce à des navigateurs web.
  2. Quelle est la condition essentielle pour votre ordinateur, pour que vous puissiez accéder au web ?
    une connexion Internet et d’un navigateur web.
  3. Où sont stockées toutes les ressources du web ?
    Les ressources du Web sont stockées sur des serveurs répartis dans le monde entier.
  4. Expliquer le processus des échanges client-serveur.
    Le client envoie une requête au serveur via Internet. Le serveur traite la demande et renvoie une réponse contenant les informations demandées. Ce processus se répète à chaque interaction.
  5. Nommer des protocoles qui régissent les échanges entre le client et le serveur.
    HTTP et HTTPS.
  6. Qu’est-ce qu’un URL ? De quoi est-il composé ?
    Un URL (Uniform Resource Locator) est l’adresse d’une ressource sur le Web. Il est composé d’un protocole (comme HTTP), d’un nom de domaine (ex: example.com) et parfois d’un chemin ou de paramètres.
  7. Qu’est-ce qu’un moteur de recherche ?
    Un moteur de recherche est un outil qui permet de trouver des informations sur le Web.
  8. Expliquer l’usage et le fonctionnement des robots Crawler ou Spider.
    Les robots Crawler ou Spider sont des logiciels qui parcourent le Web pour indexer les pages et les rendre accessibles via les moteurs de recherche.
  9. Comment un moteur de recherche propose des URL à partir de mots clés ?
    Un moteur de recherche compare les mots-clés avec sa base de données et utilise des algorithmes pour proposer une liste d’URL triée par pertinence.
  10. Qu’est-ce qu’une page web ? Que peut-elle contenir ? Nommer un langage pour en créer une.
    Une page web est un document affiché sur Internet via un navigateur. Elle peut contenir du texte, des images, des vidéos et des liens. Un langage couramment utilisé pour la créer est HTML.
  11. Comment appelle-t-on le système utilisé par le langage HTML ?
    Le langage HTML utilise un système de balisage.
  12. Qu’est-ce que le CSS ?
    C’est pour mettre en forme les pages web, comme les couleurs, les polices, les tailles, etc.
  13. Est-ce que la réutilisation d’une page Web est toujours autorisée ?
    Non, et il existe par exemple les licences Creative Commons.
