Internet et Web

  1. Que désigne ‘Le Web’ ?
    Le web, c’est toutes les ressources reliées entre elles par des liens sur lesquels on clique pour aller d’une page à une autre.
  2. Quelle est la condition essentielle pour votre ordinateur, pour que vous puissiez accéder au web ?
    Il faut que la machine soit connectée à Internet.
  3. Où sont stockées toutes les ressources du web ?
    Elles sont stockées sur des serveurs distants.
  4. Expliquer le processus des échanges client-serveur.
    Le client envoie une demande au serveur et le serveur répond en envoyant des pages web, des images, des vidéos, etc.
  5. Nommer des protocoles qui régissent les échanges entre le client et le serveur.
    HTTP et HTTPS.
  6. Qu’est-ce qu’un URL ? De quoi est-il composé ?
    Un URL, c’est une chaîne de caractères qui dit le protocole utilisé, le nom de domaine et le chemin pour accéder à la ressource.
  7. Qu’est-ce qu’un moteur de recherche ?
    C’est un outil qui donne des URL aux internautes selon ce qu’ils cherchent.
  8. Expliquer l’usage et le fonctionnement des robots Crawler ou Spider.
    Les crawlers ou spiders parcourent les liens du web pour analyser et enregistrer les contenus.
  9. Comment un moteur de recherche propose des URL à partir de mots clés ?
    Il compare les mots-clés avec sa base de données et utilise des algorithmes pour donner une liste d’URL triée par pertinence.
  10. Qu’est-ce qu’une page web ? Que peut-elle contenir ? Nommer un langage pour en créer une.
    Une page web peut avoir du texte, des images, des vidéos, des sons et des animations. On utilise le langage HTML.
  11. Comment appelle-t-on le système utilisé par le langage HTML ?
    C’est un système de balises.
  12. Qu’est-ce que le CSS ?
    C’est pour mettre en forme les pages web, comme les couleurs, les polices, les tailles, etc.
  13. Est-ce que la réutilisation d’une page Web est toujours autorisée ?
    Non, et il existe par exemple les licences Creative Commons.
